# Freely - English Vocabulary Learning App

Freelyは、英語語彙学習を楽しく継続できるReact Native（Expo）アプリです。学習活動に応じて変化する鳥のアニメーションを通じて、ユーザーの学習状況を視覚化します。

## 🐦 コアコンセプト

ユーザーの学習活動を「自由に飛ぶ鳥」として表現し、3つの軸で可視化します：

- **距離（Distance）**: アプリインストールからの経過時間（一定速度で水平移動）
- **高度（Altitude）**: 学習活動とログイン頻度に基づく（活動的なほど高く飛ぶ）
- **自由度（Freedom）**: 高度の変動度合い（学習と休息のバランス）

## 🎯 主な機能

### 学習システム
- 英検1級レベルの50単語を収録
- シンプルな「知っている」「知らない」ボタンでの学習
- 学習履歴の自動記録と進捗追跡

### 鳥のアニメーション
- **飛行（Fly）**: 高度 > 0 かつ 自由度 ≥ 0.2
- **滑空（Glide）**: 高度 > 0 かつ 自由度 < 0.2
- **跳躍（Hop）**: 高度 ≤ 0（地面での跳躍）

### データ管理
- AsyncStorageによるローカルデータ永続化
- サーバー不要の完全オフライン動作
- Zustandによる効率的な状態管理

## 🛠 技術スタック

- **Framework**: React Native (Expo)
- **State Management**: Zustand
- **Storage**: AsyncStorage
- **Animation**: React Native Reanimated
- **Graphics**: React Native SVG
- **Language**: TypeScript

## 📊 メトリクス計算

### 高度（Altitude）
- 学習活動で上昇（+5 per word）
- 非活動時間で下降（-1 per hour）
- 範囲: 0-100

### 自由度（Freedom）
- 高度履歴の分散から計算
- 適度な変動で最大値
- 範囲: 0-1

## 🔄 学習フロー

1. 単語カードが表示される
2. ユーザーが「知っている」「知らない」を選択
3. 学習記録が保存される
4. 鳥のメトリクスが更新される
5. アニメーション状態が変化する
6. 次の単語に進む
